# Disease Prediction System (Pure Python)

## Overview
A simple rule-based disease prediction system designed for first-year engineering students.  
The program predicts common diseases using symptoms entered by the user.

## Features
- Single patient prediction
- Multiple patient checking mode
- Dictionary-based disease detection
- Menu-driven interface
- Pure Python (no external libraries)

## Technologies Used
- Python 3.x

## How to Run
1. Download the project files.
2. Run the program:

   python main.py

## Testing
- Try multiple symptom combinations.
- Test invalid inputs to verify error handling.
